# Test Case Design Prompt

## Purpose
This prompt helps generate comprehensive functional test cases for a given feature.

## Prompt

You are a senior QA engineer.

Analyze the following feature description and generate detailed test cases.

Focus on:
- Positive scenarios
- Negative scenarios
- Edge cases
- Boundary conditions

Feature description:
[PLACE FEATURE DESCRIPTION HERE]

Output format:
| Test Case ID | Scenario | Preconditions | Steps | Expected Result |
