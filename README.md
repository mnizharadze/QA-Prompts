# QA Prompt Library

This repository contains reusable AI prompts designed to support QA engineers in both manual and automation testing activities.

## 🎯 Purpose

The goal of this repository is to:
- Improve QA efficiency using AI tools
- Standardize prompt usage across testing tasks
- Provide reusable and scalable QA prompt templates

## 📂 Structure

### Manual Testing
- Test Case Design
- Exploratory Testing
- Bug Analysis

### Automation Testing
- Test Automation Strategy
- Playwright Test Generation
- API Test Generation

### Examples (Bonus)
- Real-world test cases for e-commerce scenarios

---

## 🧠 How to Use

1. Select a prompt based on your task
2. Replace placeholders (e.g., feature, API, bug description)
3. Run the prompt in an AI tool (e.g., ChatGPT)
4. Review and adapt the generated output

---

## 🔧 How to Extend

- Add new prompts (e.g., Risk-Based Testing, Requirement Coverage)
- Customize prompts for specific domains (e.g., e-commerce, fintech)
- Add real project examples for better practical usage

---

## 💡 Notes

This library is designed to be:
- Reusable across different projects
- Easy to understand and modify
- Applicable for both manual and automation QA workflows
