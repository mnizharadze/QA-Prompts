# Test Automation Strategy Prompt

## Purpose
This prompt helps define an automation strategy for a feature.

## Prompt

You are a senior QA automation engineer.

Analyze the following feature and propose an automation approach.

Focus on:
- What should be automated
- What should not be automated
- Test levels (UI, API, integration)
- Test data management
- Risks and maintenance considerations

Feature description:
[PLACE FEATURE DESCRIPTION HERE]

Output format:
- Automation Scope
- Tools & Frameworks
- Test Scenarios to Automate
- Data Strategy
- Risks
